## Проверка статуса кластера
На мастер ноде, для получения состояния нод и подов:
```bash
kubectl get nodes
kubectl get pods -A
```

## Тестирование работы DNS и связи между подами
Создание тестового пода:
```bash
kubectl run -it --rm busybox --image=busybox -- sh
```

Затем, после того как контейнер будет создан, можно выполнить команду `ping`:
```bash
ping google.com -c 3
ping nfs0.lab -c 3
```

Успешное выполнение команды будет означать что DNS работает корректно
